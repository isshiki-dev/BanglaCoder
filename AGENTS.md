# BanglaCoder
Built for South Asian developers • English &amp; বাংলা supported

The AI agent platform built for South Asian developers. Create, remix, and deploy intelligent applications in English and বাংলা. 

Agent is a general-purpose agent capable of completing complex, long-horizon tasks. Supported by a multi-agent system, it can formulate expert-level solutions through multi-step planning, flexibly break down task requirements, and execute multiple sub-tasks to deliver the final result.

Firebase project is ready

### Capabilities include:

| Capabilities | Description |
| :--- | :--- |
| **Code** | **Reliably delivers full-stack web apps** with Auth, Functions, Database, and Stripe integration.<br>**Bug-free:** Ensures application stability through end-to-end testing that simulates user actions.<br>**Aesthetics:** Specializes in creating visually stunning and modern web interfaces. |
| **PPT** | **Aesthetics:** Offers flexible and beautiful layout beyond single-theme templates, with a variety of visualization components and styles.<br>**Export Quality:** State-of-the-art export quality, ensuring seamless conversion from HTML to PPTX format. |
| **Deep Research** | **Comprehensive Research:** Utilizes a wide range of tools including search, API, browser use, and MCPs to access information sources for accurate results.<br>**In-depth Analysis:** Integrates code analysis and chart generation for more comprehensive insights in research reports. |
| **Multimodal** | **Input:** Supports comprehensive understanding of long-text files, as well as videos, audio, and images.<br>**Output:** Built-in capabilities for image, audio, and video generation to create rich multimedia content. |
| **MCP Ecosystem Integration** | **Custom MCPs:** Supports the creation of any custom MCPs from scratch or by wrapping existing tools.<br>**Pre-built MCPs:** Includes a built-in MiniMax MCP and integrates with popular MCPs like Google maps, GitHub/GitLab, Slack, and Figma. |

### Features

- **Next.js App Router**
  - Advanced routing for seamless navigation and performance
  - React Server Components (RSCs) and Server Actions for server-side rendering and increased performance

- **AI SDK**
  - Unified API for generating text, structured objects, and tool calls with LLMs
  - Hooks for building dynamic chat and generative user interfaces
